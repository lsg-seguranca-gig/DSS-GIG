📋 Histórico permanente — nunca apague registros passados

            
Todos os afastamentos (passados e atuais) devem ser mantidos aqui. O sistema usa as datas de cada registro para determinar automaticamente quem estava dispensado em cada semana selecionada no Dashboard — mesmo que o período já tenha terminado. Apagar um registro passado fará o colaborador aparecer incorretamente como "Não Participou" naquela semana.

          

          

            Para cada semana selecionada no Dashboard, o sistema verifica automaticamente quais colaboradores tinham pelo menos 1 dia de afastamento naquela semana e os classifica como Dispensados (excluídos do cálculo de adesão).
